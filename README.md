# WEB103 Prework - Creatorverse

Submitted by: Ivan Lam

About this web app: Creatorverse is a web application designed for users to display their favorite content creators. Users can perform CRUD operations on a list of content creators. This project is for the CodePath WEB103 Prework.

Time spent: 3 hours
